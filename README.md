# IDAI-JaineePatel-CryptoVolatilityVisualizer
Data-driven app for analyzing cryptocurrency market stability and volatility patterns.

Overview:

The Crypto Volatility Visualizer is an interactive web-based dashboard built using Python and Streamlit that helps users explore and understand cryptocurrency price behavior. The application visualizes real crypto market data to show how prices fluctuate over time, highlighting periods of stability and high volatility. By combining mathematical concepts with data visualization, the app makes complex market movements easier to understand through clear, dynamic graphs.

Objective:

The main objective of this project is to demonstrate how mathematical functions and data analysis can be applied to real-world financial markets. The app aims to help users analyze cryptocurrency price movements, identify volatility patterns, and understand the impact of market fluctuations in an interactive and intuitive way. It also serves as a learning tool for students by connecting mathematics, programming, and financial data.

Target Audience:
Students studying Artificial Intelligence, Data Science, or Finance
Beginners who want to understand cryptocurrency market behavior
Educators looking for a visual tool to explain volatility and trends
Individuals interested in exploring crypto data without deep technical knowledge

User Persona (User Problems):
User Type: Student / Beginner Crypto Enthusiast
Key Problems-
Difficulty interpreting raw cryptocurrency price data
Limited understanding of volatility and market fluctuations
Overwhelmed by complex charts and technical financial tools
How the App Addresses These Problems-
Transforms raw data into simple, interactive visualizations
Allows users to explore price behavior using adjustable parameters
Helps users visually compare stable and volatile market periods
